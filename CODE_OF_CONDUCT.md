# 行为准则

[中文](CODE_OF_CONDUCT.md) | [English](CODE_OF_CONDUCT.en.md)

本项目希望保持直接、专业、尊重事实的协作方式。

## 我们期望的行为

- 讨论问题而不是攻击个人
- 用可复现信息支撑结论
- 尊重不同经验水平的贡献者
- 接受建设性反馈，并基于证据调整观点

## 不可接受的行为

- 人身攻击、羞辱、骚扰
- 恶意引战、刷屏、挑衅
- 故意隐瞒高风险变更影响
- 发布与项目无关的广告、政治或仇恨内容

## 执行

维护者可以对明显破坏协作秩序的内容进行编辑、隐藏、锁定或移除。严重或重复违规者可以被限制参与仓库协作。
